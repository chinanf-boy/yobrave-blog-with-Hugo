## conda

>是 Anaconda 的 命令行

因为使用**zsh**终端后，conda路径失去了

[ohmyzsh](https://github.com/robbyrussell/oh-my-zsh)

``` bash
$ conda
conda command not found
```

## 解决

```
export PATH="/home/username/miniconda/bin:$PATH"

source ~/.zshrc
```

> zsh中 .zshrc
``` js
//error:
export PATH = "/home/username/miniconda/bin:$PATH"

//right
export PATH="/home/username/miniconda/bin:$PATH"

```
> 不要保持良好的代码格式,会出错
